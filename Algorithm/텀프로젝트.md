### 풀이 과정
- 사이클이 존재한다면 사이클에 포함되는 학생들을 세서 총 학생 수 - 사이클 학생 수 를 구한다면 해결되는 문제라고 생각
- 1~N만큼 반복문을 돌면서 dfs 수행
	- 시작점부터 탐색하여 자신에게 돌아오는 순간 사이클 학생이 된다.
- 하지만 _시간초과

- 방문 배열을 2개로 만들자
	- 이미 검사가 끝난 원소를 체크(done)
	- 일반 방문 배열(visited)
- dfs를 수행
	- done == true
		- 이미 방문한 적이 있는 곳
	- done == false && visited == true
		- 방문했던 곳을 다시 방문하게 되었다. -> 사이클
		- ans++
	- done == false && visited == false
		- 재귀를 타본다.
	- 모든 재귀가 끝나면 done을 true로 만든다.
#### 다른 관점
- 이 문제를 위상정렬을 풀 듯이 풀 수 있다.
##### 조건
- 진입차수가 0이라면 사이클을 만들 수 없다.
- 해당노드와 연결되어 있는 모든 노드의 간선을 끊어줄 때 
	- 0이되지 않는다면 사이클이 존재한다는 뜻!
